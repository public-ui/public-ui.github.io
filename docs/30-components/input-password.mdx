# InputPassword

Der Input-Typ **Password** erzeugt ein Eingabefeld für Passwörter. Die Eingabe wird über Punktsymbole maskiert.

## Konstruktion

### Code

```html
<kol-input-password _label="Passwort" _smartButton='{"_icons": "codicon codicon-eye", "_hideLabel": true, "_label": "Passwort anzeigen"}'></kol-input-password>
```

### Events

Zur Behandlung von Events bzw. Callbacks siehe <kol-link _label="Events" _href="../concepts/events" />.

| Event    | Auslöser                         | Value                           |
|----------|----------------------------------|---------------------------------|
| `click`  | Eingabefeld wird angeklickt      | -                               |
| `focus`  | Eingabefeld wird fokussiert      | -                               |
| `blur`   | Eingabefeld verliert Fokus       | -                               |
| `input`  | Wert wird durch Eingabe geändert | Aktueller Wert des Eingabefelds |
| `change` | Eingabe wurde abgeschlossen      | Aktueller Wert des Eingabefelds |

### Beispiel

<kol-input-password _label="Passwort" _smartButton='{"_icons": "codicon codicon-eye", "_hideLabel": true, "_label": "Passwort anzeigen"}'></kol-input-password>

## Zeichenbegrenzung und Zähler

Die Eigenschaften `_character-limit` (über `_max-length`) und `_has-counter` arbeiten zusammen, um eine flexible Zeichenzählung und Begrenzungsfunktionalität für Eingabefelder bereitzustellen.

### Eigenschaften

- `_max-length`: Legt die maximale Zeichenanzahl für das Eingabefeld fest.
- `_max-length-behavior`: Steuert, wie die Begrenzung durchgesetzt wird:
- "hard" (Standard): Zeigt den Zähler an UND setzt das native `maxlength`-Attribut des Browsers.
- "soft": Zeigt verbleibende Zeichen an, erlaubt aber das Überschreiten der Begrenzung.
- `_has-counter`: Steuert das Anzeigeformat des Zählers:
- `true`: Zeigt die aktuelle Zeichenzahl unabhängig von einer Begrenzung an.
- `false` (Standard): Zeigt das Format "verbleibende/überschrittene Zeichen" an, wenn eine Begrenzung gesetzt ist.

### Anzeige-Modi

Der Zähler zeigt je nach Kombination der Eigenschaften unterschiedliche Formate an:

1. `_has-counter="true"` mit `_max-length`: Zeigt "5/10 Zeichen"
2. `_has-counter="true"` ohne Begrenzung: Zeigt "5 Zeichen"
3. `_max-length` mit `_max-length-behavior="soft"`: Zeigt "7 Zeichen verbleibend" oder "2 Zeichen zu viel"

### Barrierefreiheit

- Für Screenreader erfolgt die Aktualisierung des Zählers mit einer Verzögerung von 500 ms, um das Tippen nicht zu unterbrechen.
- Ein versteckter Hinweis erläutert die Begrenzung: "Sie können bis zu 10 Zeichen eingeben"
