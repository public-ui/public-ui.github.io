Datenschutzkonzept

Dieses Konzept greift die aus Sicht des Datenschutzes relevanten Aspekte der Komponenten-Bibliothek KoliBri auf und erläutert diese.

Die Datenschutzerklärung bzgl. des Open Source-Vorgehen wird in einem separaten Dokumente beschrieben.

Einleitung​

Die Komponenten-Bibliothek KoliBri hat einen ganz klaren und fokussierten Zielbereich seiner technisch funktionalen Ausprägung, die ausführlich im Manifest beschrieben wird.

KoliBri ist eine Sammlung von kleinteiligen, semantisch barrierefreien und flexibel wiederverwendbaren Web Components ohne jegliche Fachlogik oder sonstigen Daten verarbeitenden Funktionalitäten. Sie beinhaltet insbesondere keine Funktionalitäten zum Versenden von Daten. Das Ziel ist ausschließlich eine semantisch barrierefreie HTML-Auszeichnung der verschiedenen Elemente (Komponenten) in verschiedenen Styleguides sicherzustellen.
