Privacy​

We (ITZBund) and GitHub (github.io) do not collect any data about usage behavior (tracking).

For the following functionalities, data may be stored exclusively in the user's local browser:

  • Theme selection: The selected theme is stored in the browser memory (local storage).
  • Dark mode: The information whether day/night mode is selected is stored in the browser memory (local storage).
  • PWA: The website is also available as an offline variant (PWA) and can be installed locally by the user.
