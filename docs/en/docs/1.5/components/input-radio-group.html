InputRadioGroup

[DEPRECATED] Die Komponenten Input-Radio-Group und Input-Radio werden zur Komponente Input-Radio zusammengeführt. Verwendet einfach den Tag <kol-input-radio _label=""> statt <kol-input-radio-group _label="">.
